## 기본 정보

**부제**
```
면만 고르면 시간은 뽀모가 잽니다
```

**상세 설명**
```
파스타는 면 종류마다 삶는 시간이 다릅니다. 스파게티 8분, 카펠리니 2분, 리가토니 11분.
매번 봉지 뒷면을 찾아 읽고 타이머에 숫자를 옮겨 적는 대신, 면만 고르세요.

• 대표 파스타 10종의 알덴테·기본 시간을 바로 준비
• 스파게티는 굵기(1.4~1.9mm)별로 시간을 따로 안내
• 봉지에 적힌 시간이 다르면 직접 입력
• 삶는 동안 그 파스타에 어울리는 소스 레시피를 같은 화면에서 확인
• 끝나기 1분 전 예비 알림 — 면수를 미리 떠둘 수 있어요
• 다 되면 소리·진동으로 알려드립니다

물이 끓기 전 물·소금 기준부터 알려주고, 젖은 손으로도 누르기 쉽게 버튼을 크게 만들었습니다.
토마토 뽀모와 올리브 올리가 곁에서 함께 기다립니다.
```

**사용 연령**: 전체 이용가
> ⚠️ **API로 확인해보니 기본값이 `minAge: 19`(19세 이용가)로 잡혀 있다.**
> 화면에서 명시적으로 전체 이용가로 바꿔야 한다. 안 바꾸면 19세 앱으로 출시된다.

**검색 키워드**
```
파스타, 타이머, 스파게티, 알덴테, 파스타삶는시간, 면, 요리, 레시피
```

## 카테고리 및 이미지

| 항목 | 값 / 파일 |
|---|---|
| 카테고리 | 라이프스타일 (또는 푸드) — 콘솔 드롭다운에서 선택 |
| 앱 로고 (정확히 600×600 PNG) | `logo-600.png` ✅ |
| 썸네일 (정확히 1932×828 PNG) | `thumbnail-1932x828.png` ✅ |
| 세로 스크린샷 (정확히 636×1048 PNG) | `screenshot-1.png` ~ `screenshot-8.png` ✅ |

> ⚠️ 콘솔은 이미지를 **리사이징하지 않는다.** 픽셀이 1이라도 다르면 거부된다.
> 위 파일들은 2026-07-19에 규격에 맞춰 재생성해 검증까지 마쳤다.

**스크린샷 8장 — 이 순서대로 올린다** (상세 설명이 내세운 기능을 순서대로 증명)

| # | 화면 | 이 장면을 넣은 이유 |
|---|---|---|
| 1 | 홈 — 파스타 10종 | 무슨 앱인지 3초 안에 |
| 2 | 굵기·알덴테 선택 | "굵기별로 시간을 따로 안내" 근거 |
| 3 | 물·소금 안내 | 초보 실패 1순위가 시간이 아니라 물·소금 |
| 4 | 삶는 중 (05:30) | 핵심 기능 |
| 5 | 타이머 고정 + 조리 순서 | **핵심 가치** — 삶는 동안 같은 화면에서 소스 준비 |
| 6 | 1분 전 예비 알림 (00:42) | "면수를 미리 떠둘 수 있어요" 근거 |
| 7 | 완료 | 뽀모·올리 브랜드 마무리 |
| 8 | 카펠리니 2분 | 면마다 시간이 다르다는 걸 한 장으로 |

> 예비 컷은 `spare/`에 있다 — 홈 스크롤, 직접 입력, 타이머+재료.
> 콘솔 슬롯이 8칸보다 적으면 뒤에서부터(8→6) 빼면 된다. 1~5는 유지 권장.

**캡처 방법** (재생성이 필요할 때): 실제 정적 빌드를 로컬 서빙하고 Playwright로 캡처한다.
타이머 카드가 `position: sticky`라 짧은 뷰포트에선 카드 안쪽 아래가 스크롤로 도달되지 않는다.
그래서 **318×800으로 렌더링한 뒤 clip으로 318×524(=DPR2에서 636×1048)만 잘라낸다.**
진행중·완료 상태는 Playwright `page.clock`으로 시간을 빨리 감아 재현한다.

## 등록 후 Claude에게 알려줄 것

1. **appName** (등록한 값 — `pomooli`로 했다면 그대로)
2. **miniAppId** (등록하면 콘솔이 발급하는 숫자 — 이후 배포의 식별자)
3. **API 키** (배포 토큰 — 콘솔에서 발급)
4. **선택한 카테고리 이름** (대분류/중분류 — 이후 MCP로 수정할 때 필요)

이 넷을 받으면 `ait init` → 빌드 → 배포까지 Claude가 수행한다.

## ⚠️ 등록은 콘솔 웹에서 직접 해주세요 (MCP 자동 등록 금지)

Claude 쪽에 콘솔 MCP가 붙어 있어 API로도 등록이 되지만, **알려진 버그가 미수정 상태**라
자동 등록을 일부러 하지 않았다. `miniapp_create`가 **에러를 뱉으면서도 앱을 실제로 생성해버리는**
사례가 보고돼 있고([2026-07-09 제보](https://techchat-apps-in-toss.toss.im/t/mcp-bundle-upload-complete/4315)),
`appName`은 한 번 정하면 **영구히 못 바꾼다.** 잘못 쏘면 `pomooli`를 되돌릴 수 없이 날린다.

자세한 배경은 [`../apps-in-toss.md`](../apps-in-toss.md) 참고.

## 참고 — 사업자 등록은 필요 없다

[2025-12-09 공지](https://techchat-apps-in-toss.toss.im/t/topic/1666)에 따르면 사업자 등록 없이
미니앱 런칭이 가능하다. 사업자가 필요한 기능은 **토스 로그인 / 비즈월렛 / 프로모션 /
인앱 광고 / 토스 페이 / 인앱 결제** 6가지인데, 이 앱은 하나도 쓰지 않는다.

가입 요건: 만 19세 이상 + 본인 명의 토스앱 (토스 비즈니스 회원 기반).
검토 기간: 영업일 기준 1~2일.
